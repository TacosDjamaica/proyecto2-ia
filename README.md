
---

## Use

to execute the game, please run the following command:

```bash
python connect4.py <difficulty>
```

where `<difficulty>` it can be the any of this setting:

| Level   | Command  |
|---------|----------|
| Easy    |  `facil` |
| Medium  | `medio`  |
| Hard    | `dificil`|

example:

```bash
python connect4.py medio
```
